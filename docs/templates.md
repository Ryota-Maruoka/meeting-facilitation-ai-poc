# Cursor Chat テンプレート集

このファイルには、Cursor Chat で規約ファイルを使う際の**コピペ用テンプレート**を収録しています。

## 📌 使い方

1. 該当するテンプレートをコピー
2. `（作業内容）` の部分を具体的な指示に書き換え
3. Cursor Chat に貼り付けて実行

---

## 🎨 フロントエンド作業用
```

【厳守】frontend-rules.md に従って実装してください。

（作業内容）

```
